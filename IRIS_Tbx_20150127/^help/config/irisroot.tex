

    \filetitle{irisroot}{Current IRIS root folder}{config/irisroot}

	\paragraph{Syntax}

\begin{verbatim}
irisroot
X = irisroot()
\end{verbatim}

\paragraph{Output arguments}

\begin{itemize}
\itemsep1pt\parskip0pt\parsep0pt
\item
  \texttt{X} {[} char {]} - Path to the IRIS root folder.
\end{itemize}

\paragraph{Description}

The \texttt{irisroot} function is equivalent to the following call to
\href{config/irisget}{\texttt{irisget}}

\begin{verbatim}
irisget('irisroot')
\end{verbatim}
