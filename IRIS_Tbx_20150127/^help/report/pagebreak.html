pagebreak

Force page break

Syntax

P.pagebreak(Caption,...)

Input arguments

  • P [ report ] - Report object created by the report.new function.

  • Caption [ char ] - Caption for the pagebreak objet; the caption only displays in the on-screen report structure.

Generic options

See help on generic options in report objects.
