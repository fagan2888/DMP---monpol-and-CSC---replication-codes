Posterior Simulator (poster Objects)

Posterior simulator objects allow evaluating the behaviour of the posterior dsitribution, and drawing model parameters from the posterior distibution.

Posterior objects are set up within the model/estimate function and returned as the second output argument - the set up and initialisation of the posterior object is fully automated in this case. Alternatively, you can set up a posterior object manually, by setting all its properties appropriately.

Poster methods:

Constructor

  • poster - Create new empty posterior simulation (poster) object.

Evaluating posterior density

  • arwm - Adaptive random-walk Metropolis posterior simulator.
  • eval - Evaluate posterior density at specified points.
  • regen - Regeneration time MCMC Metropolis posterior simulator.

Chain statistics

  • stats - Evaluate selected statistics of ARWM chain.

Getting on-line help on model functions

help poster
help poster/function_name
